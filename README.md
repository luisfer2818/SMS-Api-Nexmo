# Node SMS Text App

Aplicativo Node.js que usa o Nexmo para enviar mensagens de texto SMS

### Version
1.0.0

### Instalação

Install the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser

```sh
$ npm start
```

## App Info

### Nexmo API Keys

Você precisará adicionar sua própria chave Nexmo api, número secreto e virtual
